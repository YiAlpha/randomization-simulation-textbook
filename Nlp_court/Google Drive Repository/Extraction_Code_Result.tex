\item

use \texttt{lexnlp} to help us extract citations from the pdf content.

website:
https://lexpredict-lexnlp.readthedocs.io/en/docs-0.1.6/index.html

    \texttt{pandas} (i.e.~Python Data Analysis Library) is a module to
manipulate data tables. After extract information from the PDF contents,
\texttt{pandas} is used to create data frame to store the information.
